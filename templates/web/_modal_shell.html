{% endblock %} Convention: the Alpine ``x-data`` on the root MUST expose a ``close()`` method — backdrop click and Esc both call it. Per-modal close logic (clear modal-root, history.replaceState, dispatch event, …) lives inside that method. ``open`` is also part of the contract for modals that need fade-out transitions (task-detail uses it; bulk-archive toggles the whole element with ``x-show``). Header actions slot (``header_actions``) sits to the left of the close button — task-detail puts its "expand to fullscreen" button there. Empty by default. {% endcomment %}
