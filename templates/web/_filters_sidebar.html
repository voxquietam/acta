{% load i18n %} {% load web_extras %} {% comment %} Shared task-filter sidebar. Used by All Tasks, My Work, and the per-project Kanban / Table page. Layout: a fixed-right floating panel on lg+. Height = content, capped at viewport. Selected values are visible at-a-glance via the active-count badge in the header (and via the page-top strip for assignee / project). Single column with sections. Section order: search → status → priority → project → labels → workspace. Project sits above labels so the long scrollable list doesn't push the small sections off the fold. Done is a regular status checkbox; the page decides whether done is hidden by default via ``default_show_done`` passed to ``apply_task_filters``. Section visibility flags: * ``filter_hide_assignee`` — drop Assignee (handled by the top strip). * ``filter_hide_workspace`` — drop Workspace (only one workspace). * ``filter_hide_project`` — drop Project (page is scoped or hoisted). * ``filter_hide_status`` — drop Status (kanban columns already group by it). {% endcomment %}
