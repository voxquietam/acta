{% load i18n %} {% load lucide %} {% comment %} Project view tabs (Overview / Kanban / Table / List). Lives in the topbar ``topbar_mid`` block on the project detail page, so the active tab is visible regardless of scroll. Clicking a tab updates the ``viewMode`` Alpine store and pushes the new ``?view=`` URL via ``history.pushState`` — no full nav, no HTMX request. The tabs' highlighted state is data-bound to the store so reverting via browser back/forward keeps the active tab in sync. Layout per ``acta-design-system/preview/comp-tabs.html`` — segmented pill: card-coloured wrapper with thin padding holds each tab; the active tab is brand-tinted with a 1px inset ring and Lucide icons. Disabled placeholders (Timeline; Overview on All Tasks) keep the slot visible with a ``lock`` glyph. {% endcomment %}
