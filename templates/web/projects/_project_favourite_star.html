{% load i18n %} {% load lucide %} {% comment %} Star toggle button for the project list. Expects ``project`` + ``is_favourite`` (bool) in context — most callers go through the ``{% project_star %}`` inclusion tag (see ``apps/web/templatetags/ web_extras.py``) which computes ``is_favourite`` from a set of favourite ids without an extra query per row. Submits to ``toggle_project_favourite``; response includes this same partial (icon flips in place) plus an OOB swap of the sidebar nav so the newly starred project appears / disappears live in the rail. ``hx-swap-oob`` is rendered ONLY in the toggle response (``oob`` flag), never on the page-rendered star. As a static attribute it would be processed during a boosted ``#app-content`` swap and — finding no existing target on the page we're navigating *to* — htmx would drop the element, so the star vanished until a full reload. {% endcomment %}
