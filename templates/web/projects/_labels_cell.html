{% load i18n %} {% comment %} Labels cell — click on the chip row (or the dashed "+" placeholder when empty) to open a dropdown listing every workspace label. Each row in the dropdown is a single-label HTMX form that toggles attachment. Multi-select pattern: the toggle endpoint swaps only the trigger contents (``#labels-trigger-inner``) and OOB-refreshes just the dropdown rows (``#labels-dropdown-inner``). The outer cell with its Alpine ``{open, q, ...}`` state stays mounted, so consecutive clicks keep the dropdown open and the search box keeps focus + its current query. The dropdown is teleported to ```` and positioned ``fixed`` from the trigger's bounding rect, so the rail's ``overflow-y-auto`` does not clip it. Flips upward when the trigger sits within ~360px of the viewport bottom. Closes on any scroll (any ancestor) to avoid the "floating popover detached from a scrolling trigger" gotcha — same UX as a native ``
