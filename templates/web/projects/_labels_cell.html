{% load i18n %} {% comment %} Labels cell — click on the chip row (or the dashed "+" placeholder when empty) to open a dropdown listing every workspace label. Each row in the dropdown is a single-label HTMX form that toggles attachment. Multi-select pattern: the toggle endpoint swaps only the trigger contents (``#labels-trigger-inner``) and OOB-refreshes just the dropdown rows (``#labels-dropdown-inner``). The outer cell with its Alpine ``{open, q, ...}`` state stays mounted, so consecutive clicks keep the dropdown open and the search box keeps focus + its current query. Dropdown direction flips upward when the cell sits near the bottom of the viewport (under ~360px of space below), because the rail's ``overflow-y-auto`` clips fixed-direction popovers at the bottom edge. Includes a client-side search input filtering labels by name substring (same UX as the assignee picker). Empty trigger uses a dashed-circle placeholder + muted text matching the unassigned assignee treatment. {% endcomment %}
