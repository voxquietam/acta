{% load i18n %} {% comment %} Static toolbar above the TipTap editor (always visible while the editor is mounted). Drop this partial inside any element marked ``[data-description-editor]`` and the JS module wires it up to the editor in that container. We swapped from a tippy-positioned bubble menu to a static toolbar after the bubble's show/hide kept misfiring in edge cases — see ``feedback_stop_kostyling`` memory for the lesson learned. Two modes via Alpine state — "menu" (default toolbar) and "link" (an inline URL input replacing the buttons, so we don't fall back to the native ``window.prompt`` dialog on link insertion). {% endcomment %}
