{% load i18n %} {% comment %} Inline edit composer for one task comment. Loaded via ``hx-get`` into the comment's ``#comment-body-`` region (innerHTML), so the header above (avatar / author / date) stays put and the card barely shifts — only the body turns editable (Linear-style). The editor mounts on ``htmx:afterSwap``, pre-filled from the Markdown body via the ``.description-editor-source`` textarea. Save posts to ``edit_comment`` and replaces the whole card (``#comment-`` outerHTML) with the refreshed render (header then shows the ``(edited)`` marker). Cancel re-fetches the unchanged card via ``comment_fragment`` (also tearing the editor down cleanly). Expects ``task`` + ``comment``. {% endcomment %}
