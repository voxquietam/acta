{% load i18n %} {% load lucide %} {% comment %} One tab button inside the workspace settings ``topbar_mid`` strip. Click updates Alpine's ``tab`` (the strip's own), pushes ``?tab=`` so the URL stays shareable, and dispatches ``acta:settings-tab`` on ``window`` so the main-content ``x-data`` flips its ``x-show`` panels in lockstep. Required context: key — tab id (``general`` / ``workflow`` / ``danger``). icon — lucide icon name shown left of the label. label_text — visible label (English; ``{% trans %}``ed at render). tone — active-state colour. ``brand`` (everyday) or ``rose`` (destructive — Danger tab). {% endcomment %}
